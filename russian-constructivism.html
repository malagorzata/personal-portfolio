RUSSIAN CONSTRUCTIVISM

TASK

TASK

The idea of this assignment was to create our first mobile-only responsive website. This included testing the use of html semantic tags, subheadings, headers as well as applying Gestalt laws. Additionially, the task involved getting basic knowledge about typography, image formats and image editing.

PROCESS

PROCESS

The first step of the task was to divide the given text and make it readable according to the learnt rules. Secondly, I edited and resized images to make them suitable for the website. The most important lesson was to apply HTML and CSS to make the website more interesting and learn how to use FileZilla, code validation websites and Brackets.

RESULT

RESULT

In this assignment, I have learnt the basic rules of HTML and CSS as well as design laws. I chose fonts that would fit the subject of Russian Constructivism. I differentiated between font weights to highlight the importance of different sections. Additionially, through application of Gestalt Laws, I could utilise the principles to make readable and user-friendly website. Especially Past Experience principle was important as I used colours and shapes that could indicate the direct link to the discussed subject. Through usage of semantic mark-ups, I could divide the text in a professional manner.
